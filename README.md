# Aplikasi Mobile Banking BRImo (Flutter)

Project ini dibuat untuk memenuhi tugas UTS Mobile Programming.

Fitur:
- Tampilan saldo
- Menu utama (Transfer, BRIVA, dll)
- Menu tambahan
- Bottom Navigation (Home, Mutasi, QRIS, dll)

Dibuat menggunakan Flutter dengan konsep widget:
- Scaffold
- AppBar
- Column & Row
- Container
